# Contributing to ds-ai-journal

Thank you for considering contributing to **ds-ai-journal**! Whether you are updating a notebook, fixing an analytical example, or expanding explanations, your input is welcome.

---

##  Code of Conduct

This project adheres to the standard [Contributor Covenant](https://www.contributor-covenant.org/). By participating, you are expected to uphold a welcoming, respectful, and collaborative environment.

---

##  How to Contribute

### 1. Suggesting Improvements
Before submitting large changes, please open an issue to discuss what you'd like to update or add.

### 2. Submitting a Pull Request
1. Fork this repository.
2. Clone your fork locally using `git clone https://github.com/mitalidaduria/ds-ai-journal.git`.
3. Create a feature branch using `git checkout -b docs/update-guidelines`.
4. Commit your changes using `git commit -m "docs: add contribution guide"`.
5. Push to your fork using `git push origin docs/update-guidelines`.
6. Open a Pull Request on GitHub.

---

##  Welcome Contributions

* **Notebook Fixes & Updates:** Correcting code syntax, improving explanations, or updating library versions.
* **Documentation & Guides:** Clarifying setup steps, adding references, or improving Markdown readability.
