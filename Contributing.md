# Contributing to ai-ba-resources

Thank you for considering contributing to **ai-ba-resources**! Whether you are adding a template, correcting an existing spec, or sharing a worked example, your input helps make this a better toolkit for everyone in the Data & Product community.

---

## Code of Conduct

This project adheres to the standard [Contributor Covenant](https://www.contributor-covenant.org/). By participating, you are expected to uphold a welcoming, respectful, and collaborative environment.

---

## How to Contribute

### 1. Suggesting New Templates or Features
Before creating a large Pull Request, please **open an issue** first to discuss your idea.
* Go to the **Issues** tab on GitHub.
* Click **New Issue**.
* Briefly describe the template, resource, or fix you would like to add and why it is useful.

### 2. Submitting a Pull Request (PR)
When you are ready to submit changes:
1. **Fork** this repository to your GitHub account.
2. **Clone** your fork locally:
   ```bash
   git clone [https://github.com/YOUR-USERNAME/ai-ba-resources.git](https://github.com/YOUR-USERNAME/ai-ba-resources.git)
